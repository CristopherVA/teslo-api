# Teslo APi
1. CLonar proyecti
2. ```yarn install o npm install```.
3. CLonar el archivo ```env.example``` y renombrarlo a  ```.env``` 
5. Levantar la base de datos
```
  docker-compose up -d
```
6. Levantar: ```yarn start:dev```
